# Weather Dashboard Pro v5.2

## Highlights

- Overview tab with saved-location highlights
- Detailed tab with one-location-at-a-time carousel controls
- Wet Bulb tab with calculator and interpretation guide
- Labeled hourly cards
- Outdoor Score
- Rain timeline
- Best dry window
- Activity guidance
- GitHub Pages workflow that avoids lock-file registry issues

This package intentionally does not include `package-lock.json`. The workflow uses `npm install --registry=https://registry.npmjs.org/` and does not enable npm cache.
